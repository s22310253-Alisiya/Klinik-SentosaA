Panduan Penggunaan Sistem Klinik Sentosa
1. Login ke Sistem
•	1. Npm install lalu Npm run dev
•	Setalah itu vite akan menampilakan alamat lokal biasanya : Local:   http://localhost:5173/
•	 Klik itu atau buka manual di browser 
•	2. Masukkan username dan password sesuai role (Admin, Resepsionis, Dokter, Apoteker).
•	3. Klik tombol Login untuk masuk ke dashboard.
2. Fitur untuk Resepsionis
•	• Pendaftaran Pasien Baru: masuk ke menu Pasien → Tambah Pasien → isi data → Simpan.
•	• Pendaftaran Pemeriksaan: menu Pendaftaran → Daftar Pemeriksaan 
•	• Mengelola Antrian: menu Antrian untuk melihat pasien yang menunggu.
3. Fitur untuk Dokter
•	• Melihat Antrian Pasien: menu Pemeriksaan.
•	• Mengisi Rekam Medis: isi keluhan, diagnosa, tindakan.
•	Pilih  nama dokter 
•	• Membuat Resep: tambah obat lalu kirim ke apoteker.
4. Fitur untuk Apoteker
•	• Melihat Resep Masuk: menu Resep Obat.
•	• meliat data obat dan tambah obat
•	• Menyerahkan Obat: konfirmasi obat dan selesaikan transaksi.
5. Fitur untuk Admin
•	• Mengelola Data User: tambah/ubah/hapus user (dokter, resepsionis, apoteker).
•	• Mengelolah semua fitur yang ada 
6. Laporan Sistem
•	• Laporan kunjungan, rekam medis, transaksi, dan obat.
•	• Cara akses: menu Laporan → pilih jenis laporan → pilih tanggal → cetak/export PDF.
7. Logout
•	Klik tombol Logout di pojok kiri bawa untuk keluar dari sistem.
